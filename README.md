# Daily Expense Tracker

A simple Python application to track and manage daily expenses.

## Features

- Add expenses with description and amount
- View all recorded expenses
- Calculate total expenses

## Usage

```python
# Add expenses
add_expenses("Lunch", 95.80)
add_expenses("Dinner", 20.80)

# View all expenses
view_expenses()

# Display total
total_expenses()
```

## Output

```
Lunch: $95.80
Dinner: $20.80
Total Expenses: $116.60
```

## Functions

- `add_expenses(description, amount)` - Add a new expense
- `view_expenses()` - Display all expenses
- `total_expenses()` - Show total amount spent
